# 🚀 快速启动指南

## 3 步启动

### 1️⃣ 安装依赖

```bash
npm install
```

### 2️⃣ 启动开发服务器

```bash
npm run dev
```

### 3️⃣ 访问应用

打开浏览器：`http://localhost:3000`

---

## 📱 使用流程图

```
首次访问
    ↓
自动跳转到 /setup
    ↓
创建事项（填写名称、时间、密码）
    ↓
可选：配置 GitHub 同步
    ↓
进入主界面 /main
    ↓
开始录入礼金
    ↓
实时展示在右侧
    ↓
导出 Excel / 打印 PDF / 开启副屏
```

---

## 🎯 核心功能演示

### 录入礼金
1. 左侧填写：姓名、金额
2. 选择类型：现金/微信/支付宝/其他
3. 可选：添加备注
4. 点击"确认录入"或按回车

### 查看数据
- 右侧实时显示 12 格礼簿
- 自动计算金额大写
- 支持翻页查看历史

### 导出数据
- **Excel**：点击"导出Excel"按钮
- **PDF**：点击"打印/PDF"，使用浏览器打印功能
- **副屏**：点击"开启副屏"，新窗口展示

---

## 💡 小技巧

### 快捷键
- **回车**：在表单中按回车自动提交
- **Tab**：快速切换输入框
- **F11**：全屏模式（副屏推荐）

### 数据安全
- 首次使用请牢记管理密码
- 定期导出 Excel 备份
- 如使用 GitHub，确保 Token 安全

### 副屏使用
1. 点击"开启副屏"
2. 将新窗口拖到副屏
3. 按 F11 全屏
4. 主屏继续录入，副屏实时更新

## 🆘 遇到问题？

### 无法启动
```bash
# 1. 清理并重装
rm -rf node_modules package-lock.json
npm install

# 2. 检查 Node 版本
node --version  # 需要 18+

# 3. 检查 npm 版本
npm --version   # 需要 9+
```

### 页面白屏
- 打开浏览器控制台 (F12)
- 查看错误信息
- 检查网络连接

### 数据丢失
- 检查 localStorage 是否被清理
- 如使用 GitHub，检查仓库数据
- 从 Excel 备份恢复
