## 📖 使用说明

### 首次使用

1. 打开 `index.html`
2. 点击"创建新事件"
3. 填写活动名称、时间
4. 设置密码（默认 `123456`）
5. 选择主题颜色

### 录入礼金

1. 点击"录入礼金"
2. 输入姓名和金额
3. 自动显示中文大写金额
4. 保存即可

### 查看统计

- 自动计算总金额
- 显示人数统计
- 支持搜索和筛选

### 导出备份

- **打印**：生成精美纸质版
- **Excel**：导出表格存档
- **副屏**：现场大屏展示

## ❓ 常见问题

**Q: 需要安装软件吗？**
A: 不需要！只有一个 HTML 文件，双击就能用。

**Q: 数据安全吗？**
A: 非常安全。数据加密后存储在本地浏览器，不会上传到任何服务器。

**Q: 能联网同步吗？**
A: 目前只支持本地使用。如需多设备同步，可以手动导出导入数据。

**Q: 会收费吗？**
A: 永久免费，开源项目。

**Q: 支持哪些浏览器？**
A: Chrome、Firefox、Edge、Safari 等现代浏览器都支持。
