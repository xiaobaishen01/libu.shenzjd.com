## 📦 版本更新

### v1.0.2 - 副屏自动轮播优化

**🎯 核心改进：**
- ✅ **副屏自动滚动**：新数据录入自动滚动到最新记录
- ✅ **10秒等待机制**：展示最新数据10秒
- ✅ **单向无限轮播**：超过10秒无新数据，自动从右向左滚动展示所有数据
- ✅ **实时中断**：有新数据立即停止轮播，重新开始循环

**🔄 工作流程：**
```
新数据录入 → 自动滚动到最新 → 停留10秒 → 超时后单向轮播 → 新数据中断 → 重新开始
```

**🎨 打印功能优化：**
- ✅ **横屏A4**：充分利用纸张空间
- ✅ **全量打印**：一次性打印所有礼金记录
- ✅ **彩色主题**：喜事红色/丧事灰色自动适配
- ✅ **竖排格式**：传统礼簿风格

### v1.0.1 - 数据备份与恢复

**💾 全新备份系统：**
- ✅ **一键导出**：下载 JSON 备份文件

- ✅ **多设备同步**：设备A导出 → 设备B导入

## 📖 使用说明

### 快速开始

**首次使用：**
1. 下载 `index.html`
2. 双击打开
3. 选择：✨ 创建新事件 或 📂 导入备份

**录入礼金：**
1. 点击"录入礼金"
2. 输入姓名和金额
3. 自动显示中文大写金额
4. 保存即可

### 副屏功能（投影展示）

**开启副屏：**
1. 主界面点击"开启副屏"
2. 自动打开最大化窗口
3. 连接投影仪或副屏

**自动轮播：**
- 新数据录入 → 自动滚动到最新
- 停留10秒 → 展示最新记录
- 超时后 → 从右向左单向轮播所有数据
- 有新数据 → 立即中断，重新开始

### 打印功能

**打印礼簿：**
1. 主界面点击"打印/PDF"
2. 自动打开新窗口（横屏A4）
3. 生成彩色内容（根据主题）
4. 自动调用打印对话框

### 数据备份

**导出备份：**
1. 主界面点击 💾 导出备份
2. 保存 .json 文件

**导入备份：**
1. 首页或主界面点击 📂 导入备份
2. 选择 .json 文件
3. 智能合并，自动跳过重复

**多设备同步：**
1. 设备A：导出备份 → 发送文件（微信/QQ/邮件）
2. 设备B：导入备份 → 自动加载所有数据

## ❓ 常见问题

**Q: 需要安装软件吗？**
A: 不需要！只有一个 HTML 文件，双击就能用。

**Q: 数据安全吗？**
A: 非常安全。数据加密后存储在本地浏览器，不会上传任何服务器。

**Q: 数据会丢失吗？**
A: 浏览器清除数据会丢失。务必使用"导出备份"功能定期备份！

**Q: 副屏如何同步？**
A: 主屏录入后，副屏2秒内自动更新并滚动到最新记录。

**Q: 会收费吗？**
A: 永久免费，开源项目。
