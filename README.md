## ✨ 核心特性

### 🔍 智能搜索与筛选

- **快速搜索**：按姓名或备注实时搜索，支持模糊匹配
- **类型筛选**：按支付方式筛选（现金/微信/支付宝/其他）
- **时间排序**：支持正序/倒序查看记录
- **实时统计**：筛选结果自动统计，一目了然
- **一键清空**：快速重置所有筛选条件

### 💾 数据备份与恢复

- **Excel导入**：支持导入模板，自动识别重复数据
- **冲突处理**：跳过/覆盖/都保留三种策略
- **模板下载**：提供标准Excel模板，填写更便捷
- **一键导出**：随时导出Excel备份数据

### 🖥️ 副屏展示（投影专用）

- **实时同步**：主屏录入，**即时更新**（使用BroadcastChannel API）
- **智能优化**：数据变化检测，避免无效刷新
- **大字体显示**：标题和数据字体放大，远距离清晰可见
- **单行展示**：显示最新的12条记录，无空白区域
- **最新高亮**：最后一条记录脉冲动画高亮
- **高度自适应**：完美填充屏幕，视觉舒适
- **主题适配**：喜事红色 / 丧事灰色

### 📄 打印存档

- **横屏A4**：充分利用纸张空间
- **全量打印**：一次性打印所有礼金记录
- **彩色主题**：喜事红色 / 丧事灰色自动适配
- **竖排格式**：传统礼簿风格，符合中式习惯
- **自动统计**：包含总金额、人数、支付方式统计

### 日常操作

**录入礼金：**

1. 点击"录入礼金"
2. 输入姓名和金额
3. 自动显示中文大写金额
4. 保存即可
5. ✅ 自动语音播报确认信息（如：`张三，贰仟元整，现金`）

**搜索与筛选：**

- 🔍 **搜索框**：输入姓名或备注关键词，实时过滤
- 📋 **类型筛选**：按支付方式筛选（现金/微信/支付宝/其他）
- ⏰ **时间排序**：选择正序或倒序查看
- 📊 **实时统计**：筛选结果自动统计显示
- 🔄 **一键清空**：快速重置所有筛选条件

**数据备份：**

- 📊 **导出Excel**：一键导出所有礼金记录
- 📥 **导入Excel**：支持导入模板，自动合并数据
- 📋 **下载模板**：提供标准Excel模板，方便填写

**开启副屏（投影）：**

1. 主界面点击"开启副屏"
2. 自动打开最大化窗口
3. 连接投影仪或副屏展示

## 💡 小贴士

- 💾 **定期备份**：务必定期导出备份，防止浏览器数据丢失
- 🎙️ **语音播报**：无需看屏幕，听声音确认录入，适合忙碌场合
- 🔍 **智能搜索**：宾客多时，用搜索快速定位记录

- 📄 **打印存档**：活动结束后打印保存珍藏
- 🔄 **多设备同步**：导出Excel → 发送文件 → 导入Excel，轻松同步
- 📊 **Excel统一**：导入导出都用Excel，简单方便
- 🔊 **浏览器权限**：首次使用可能需要点击允许语音播报

## ❓ 常见问题

**Q: 需要安装软件吗？**
A: 不需要！只有一个 HTML 文件，双击就能用。

**Q: 数据安全吗？**
A: 非常安全。数据存储在本地浏览器，不会上传任何服务器。

**Q: 数据会丢失吗？**
A: 浏览器清除数据会丢失。务必使用"导出备份"功能定期备份！

**Q: 副屏如何同步？**
A: 主屏录入后，副屏**即时更新**（使用BroadcastChannel API，比2秒更快）。

**Q: 语音播报怎么用？**
A: 录入礼金后会自动播报，无需设置。支持Chrome、Edge、Firefox、Safari。

**Q: 语音播报不工作怎么办？**
A: 1) 检查浏览器是否支持；2) 首次使用可能需要点击允许语音权限；3) 确保浏览器音量开启

**Q: 会收费吗？**
A: 永久免费，开源项目。

**Q: 有搜索筛选功能吗？**
A: 有！支持按姓名/备注搜索，按支付方式筛选，按时间排序，实时统计。

**Q: 数据量大了会卡吗？**
A: 不会。已优化性能，支持分页显示，搜索筛选快速响应。
