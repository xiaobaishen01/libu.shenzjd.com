# 📦 发布指南

**✨ 只有这一个文件：index.html**

下载后双击即可使用，无需解压，无需安装。

---

## 🚀 如何发布

### 方式1: Git Tag（推荐）

```bash
git tag v1.0.0
git push origin v1.0.0
```

GitHub Actions 自动完成：构建 → 发布 → 上传 index.html

### 方式2: 手动触发

1. GitHub → Actions → Release 工作流
2. 点击 **Run workflow**
3. 输入版本号（如：v1.0.0）
4. 点击运行
