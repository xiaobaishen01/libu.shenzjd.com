# 🚀 启动说明 - 电子礼簿系统

## ⚡ 最快启动方式

### 方式 1：使用启动脚本（推荐）

**Mac/Linux:**
```bash
./start.sh
```

**Windows:**
```bash
start.bat
```

### 方式 2：手动启动

```bash
# 1. 安装依赖（如果还没安装）
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问应用
# 打开浏览器访问：http://localhost:3000
```

---

## ✅ 启动成功标志

你应该看到类似以下输出：

```
✓ Ready in 3.2s
○ Compiling / ...
✓ Compiled / in 2s
- Local:    http://localhost:3000
```

然后浏览器会自动打开，显示：
- **首次使用**：事项创建页面
- **已有事项**：主界面

## 📱 页面说明

### 首页 (`/`)
自动检测并跳转

### 事项创建 (`/setup`)
- 填写基本信息
- 可选 GitHub 配置
- 创建后自动进入主界面

### 主界面 (`/main`)
- **左侧**：录入表单
- **右侧**：礼簿展示（12格/页）
- **顶部**：导出/副屏按钮
- **底部**：分页导航

### 副屏 (`/guest-screen`)
- 独立窗口
- 实时同步
- 适合投屏

## 🔧 常见问题

### 问题：端口被占用
```bash
# 修改端口
npm run dev -- -p 3001
```

### 问题：依赖安装失败
```bash
# 清理并重试
rm -rf node_modules package-lock.json
npm install
```

### 问题：TypeScript 错误
```bash
# 检查类型
npx tsc --noEmit
```
